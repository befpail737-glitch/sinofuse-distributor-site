### 🆘 Troubleshooting

#### Problem: Page returns 404
**Solution**: Check file uploaded to correct directory

#### Problem: Schema not detected
**Solution**: 
1. Check JSON syntax: https://jsonlint.com/
2. Verify script is in `<head>` section
3. Test with Rich Results Test

#### Problem: Pages not indexing
**Solution**:
1. Check robots.txt allows crawling
2. Verify noindex meta tag not present
3. Request indexing in GSC
