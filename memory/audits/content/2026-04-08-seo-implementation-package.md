## Schema Markup (JSON-LD)

### Organization Schema
- 包含品牌信息、联系方式、服务地区
- 支持多语言：English, German, Vietnamese, Hindi, Japanese, Korean

### LocalBusiness Schema
- 越南总部仓库信息
- 支持本地搜索可见性

### FAQ Schema
- 4 个常见问题
- 支持 SERP 精选摘要

### Product Schema
- 4 个产品类别
- 支持产品轮播展示
