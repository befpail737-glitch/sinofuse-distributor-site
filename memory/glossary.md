## 项目特定术语

### 品牌与产品
| 术语 | 定义 |
|------|------|
| Sinofuse | 电子元器件品牌，主营保险丝、电路保护器件 |
| Fuse | 保险丝，过流保护器件 |
| Circuit Protection | 电路保护，包括保险丝、断路器等 |

### 业务术语
| 术语 | 定义 |
|------|------|
| Authorized Distributor | 授权代理商，经品牌官方认证的分销商 |
| Local Warehouse | 本地仓库，位于目标市场的仓储设施 |
| Local Delivery | 本地交货，从本地仓库发货的配送服务 |
| Authorization Certificate | 授权证书，品牌颁发的分销授权证明文件 |

### SEO/GEO 术语
| 术语 | 定义 |
|------|------|
| GEO SEO | 地理定位搜索引擎优化，针对特定国家/地区的 SEO |
| Topic Cluster | 主题集群，围绕核心主题组织的内容结构 |
| Hero Keywords | 核心关键词，优先级最高的目标关键词 |
| Long-tail Keywords | 长尾关键词，搜索量较低但转化率高的具体关键词 |
| EEAT | Experience, Expertise, Authoritativeness, Trustworthiness |
| CITE | Citation, Intent, Trust, Engagement (域名评分体系) |

### 市场术语
| 术语 | 定义 |
|------|------|
| P1 Market | 优先级 1 市场 (美国、德国、越南、印度) |
| P2 Market | 优先级 2 市场 (韩国、意大利、波兰、俄罗斯、马来西亚、巴西) |
| DA | Domain Authority，域名权威度 |
| Local Search Visibility | 本地搜索可见性，在目标地区搜索结果中的曝光度 |
